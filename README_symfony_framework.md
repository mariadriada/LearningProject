		TEST 2: SYMFONY FRAMEWORK - Configuration 
/////////////////////////////////////////////////

1. Download file in project directory:

	 php -r "readfile('http://symfony.com/installer');" > symfony.phar

2. Install symfony:

	php symfony.phar

3. Create project:
	
	php symfony.phar new my_project_name

4. Run project:
	
	php app/console server:run
